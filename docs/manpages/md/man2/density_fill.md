# NAME

density_fill - density fill

# SYNOPSIS

density_fill
    [-rules rules_file]
    [-area {lx ly ux uy}]


# DESCRIPTION

This command performs density fill to meet metal density DRC rules.

# OPTIONS

`-rules`:  Specify `json` rule file.

`-area`:  Optional. If not specified, the core area will be used.
