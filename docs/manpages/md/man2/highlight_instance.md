gui::highlight_inst 
       name
       [highlight_group]


# DESCRIPTION

To highlight an instance:

# OPTIONS

`name`:  name of the instance to highlight.

`highlight_group`:  group to add the highlighted instance to, defaults to ``0``, valid groups are ``0 - 7``.
