# NAME

report_floating_nets - report floating nets

# SYNOPSIS

report_floating_nets 
    [-verbose]


# DESCRIPTION

The `report_floating_nets` command reports nets with only one pin connection.

# OPTIONS

`-verbose`:  Print the net names.
