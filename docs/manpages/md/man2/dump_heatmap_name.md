dump_heatmap_name - dump heatmap name

# SYNOPSIS

gui::dump_heatmap 
       name 
       filename


# DESCRIPTION

To save the raw data from the heat maps ins a comma separated value (CSV) format:

# OPTIONS

`name`:  is the name of the heatmap.

`filename`:  path to the file to write the data to.
