# NAME

use_interface_cell - use interface cell

# SYNOPSIS

use_interface_cell
    [-domain domain]
    [-strategy strategy]
    [-lib_cells lib_cells]


# DESCRIPTION

This command sets the interface cell.

# OPTIONS

`-domain`:  Power domain name.

`-strategy`:  Isolation strategy name.

`-lib_cells`:  List of lib cells that could be used.
