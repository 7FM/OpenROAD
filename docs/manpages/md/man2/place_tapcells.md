# NAME

place_tapcells - place tapcells

# SYNOPSIS

place_tapcells
    -master tapcell_master
    -distance dist


# DESCRIPTION

This command is used for tapcell placement only.

# OPTIONS

`-master`:  Master to use for the tapcells.

`-distance`:  Distance between tapcells.

# ARGUMENTS

This command has no arguments.
